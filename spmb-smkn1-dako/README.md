# spmb-smkn1-dako

Struktur proyek sederhana untuk halaman pengumuman SPMB SMKN1 Dako.

Struktur:

- `index.html` - halaman utama
- `css/style.css` - stylesheet
- `js/` - skrip frontend (app, countdown, api, search, ui)
- `gas/Code.gs` - Google Apps Script (opsional)
- `images/` - simpan `logo.png` sekolah di sini

Cara menjalankan cepat:

1. Buka `index.html` di browser.
2. Tempatkan logo sekolah di `images/logo.png`.
